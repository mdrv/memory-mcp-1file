# VIDA: Feature Checklist Management

Read and execute the VIDA feature command instructions:

@vida/commands/vida-feature.md

Use @AGENTS.md for Memory Protocol context.

---

## Commands

| Command | Description |
|---------|-------------|
| `/project:vida-feature` | Analyze feature checklist coverage |
| `/project:vida-feature analyze` | Same as above |
| `/project:vida-feature link` | Show feature → spec → epic mappings |
| `/project:vida-feature add <feature>` | Add new feature to checklist |
| `/project:vida-feature update` | Sync feature states from specs |
| `/project:vida-feature coverage` | Show detailed coverage report |

## Purpose

Managing business features and their relationship to specifications:
1. Analyze the feature checklist for coverage
2. Link features to specifications
3. Identify features needing specs
4. Track feature → spec → epic relationships
